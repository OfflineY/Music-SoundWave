关于此项目

这是一个音乐库，他将会存储一些歌曲，你可以对他们进行删、改、查等操作。前端界面为 Html+Js(jquery)+css 完成，页面借鉴网易云音乐(music.163.com)的设计，然后再加上一些删改完成的，后端待建设，预计使用 MongoDB数据库+Express 完成。此项目开源在 Github ，使用许可证 MIT 。
